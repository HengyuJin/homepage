# Hengyu Jin Personal Homepage

This is a lightweight academic and professional personal homepage built with
plain HTML and CSS. It is designed for job applications, graduate applications,
research collaboration, and personal branding.

## Preview Locally

Open `index.html` directly in a browser, or run a small local server:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Deploy With GitHub Pages

1. Create a new GitHub repository, for example `homepage`.
2. Push this folder to the repository.
3. In GitHub, open the repository settings.
4. Go to `Pages`.
5. Choose `Deploy from a branch`.
6. Select the `main` branch and `/root` folder.
7. Save and wait for GitHub to publish the site.

Your public link will look like:

```text
https://YOUR-GITHUB-USERNAME.github.io/homepage/
```

If the repository is named `YOUR-GITHUB-USERNAME.github.io`, the public link will
look like:

```text
https://YOUR-GITHUB-USERNAME.github.io/
```
